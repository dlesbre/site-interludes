Pendant les 48h des jeux, les clubs jeux (de société, de carte, de rôle, vidéo, DDR...) de l'ENS Ulm se retrouvent pour un week-end ludique. Elles se déroulent chaque année fin septembre/début octobre. Rejoins-nous au Pôt pendant ces deux jours magiques où le jeu sous toutes ses formes est à l'honneur !

Cette année, elles sont prévues {% if settings.date_start %}{% if settings.date_end %} {% if settings.date_end.month != settings.date_start.month %} le week-end du {{ settings.date_start|date:"j F" }}-{{ settings.date_end|date:"j F Y" }}. {% else %} le week-end du {{ settings.date_start.day }}-{{ settings.date_end }}. {% endif %} {% else %} le week-end du {{ settings.date_start }}. {% endif %} {% else %} à une date non-fixée. {% endif %}

L'événement est entièrement gratuit et se déroule principalement au Pôt (restaurant). Voir la FAQ pour plus de détails.

Activités

En plus des jeux disponibles en continu au Pôt, vous pouvez trouver sur ce site les activités plus spécifiques proposées lors des 48h des jeux. Certaines nécessitent une inscription (car nombre de places limitées ou nécessitant une préparation). Si c'est le cas, vous pouvez contacter l'orga par email pour vous y inscrire.

Pour plus de détails, consultez la FAQ.
