Présentation

Pendant les 48h des jeux, les clubs jeux (de sociéte, de carte, de rôle, vidéo, DDR...) de l'ENS Ulm se retrouvent pour un week-end ludique. Elles se deroulent chaque année fin septembre/début octobre. Rejoins-nous au Pôt pendant ces deux jours magiques où le jeu sous toutes ses formes est à l'honneur !

Cette année, elles sont prévues {% if settings.date_start %}{% if settings.date_end %} le week-end du {{ settings.date_start.day }}-{{ settings.date_end }}. {% else %} le week-end du {{ settings.date_start }}. {% endif %} {% else %} à une date non-fixée. {% endif %}

La situation actuelle impose des mesures spéciales pour l'événement, notamment le port du masque et le contrôle du passe sanitaire. Voir la FAQ pour plus de détails.

Activités

L'événement est entièrement gratuit.

En plus des jeux disponibles en continue au Pôt, vous pouvez trouver sur ce site les activités plus spécifiques proposées lors des 48h des jeux. Certaines nécessitent une inscription (car nombre de places limités ou nécessitant une préparation). Si c'est le cas, vous pouvez contacter l'orga par email pour vous y inscrire.

Pour plus de détails, consultez la FAQ.
