Présentation

Les 48h des jeux sont le week-end de rentré des clubs jeux, jeux vidéo et jeu de rôle de l'ENS Ulm. Elles se deroulent chaque année fin septembre/début octobre. Rejoins-nous aux Pôt pendant ce week-end magique où les jeux de plateau, jeux de rôles, jeux vidéos, murders et autres sont à l'honneur.

Cette année, elles sont prévues {% if settings.date_start %}{% if settings.date_end %} le week-end du {{ settings.date_start.day }}-{{ settings.date_end }}. {% else %} le week-end du {{ settings.date_start }}. {% endif %} {% else %} à une date non-fixée. {% endif %}

La situation actuelle impose des mesures spéciales pour l'événement, notamment le port du masque et le contrôle du pass sanitaire. Voir la FAQ pour plus de détails.

Inscriptions

  • Les inscriptions sont facultatives. Elles servent uniquement pour les activités avec préparation/au nombre de place limité.

Ce site vous permettra de vous inscrire à l'événement et de voir les activités qui y seront proposées. L'événement est gratuit.
